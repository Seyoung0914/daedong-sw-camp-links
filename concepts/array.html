Lesson 07

배열

배열은 같은 자료형의 여러 값을 하나의 이름으로 묶어 번호를 붙여 관리하는 문법입니다.

이전 개념다음 개념

핵심 설명

배열을 만들 때는 자료형, 배열 이름, 크기를 적습니다. 예를 들어 int scores[5];는 정수 5개를 저장할 수 있는 배열입니다.

배열의 번호는 0부터 시작합니다. 크기가 5인 배열의 인덱스는 0부터 4까지입니다.

기억할 점

  • 배열 범위를 벗어난 인덱스에 접근하면 예상하기 어려운 문제가 생깁니다.
  • 반복문을 사용하면 배열의 값을 차례대로 처리하기 쉽습니다.
  • 배열 이름은 배열의 시작 주소처럼 동작하는 경우가 많습니다.

예제 코드

#include <stdio.h>

int main(void) {
    int scores[5];
    int sum = 0;

    for (int i = 0; i < 5; i++) {
        printf("%d번째 점수: ", i + 1);
        scanf("%d", &scores[i]);
        sum += scores[i];
    }

    printf("총점: %d\n", sum);
    printf("평균: %.1f\n", sum / 5.0);
    return 0;
}
