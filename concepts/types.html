Lesson 02

변수와 자료형

변수는 값을 담는 이름 붙은 공간이고, 자료형은 그 공간에 어떤 종류의 값을 담을지 정합니다.

이전 개념다음 개념

변수는 뭐야?

변수는 값을 저장하는 상자입니다. 예를 들어 나이를 저장하고 싶으면 int age = 20;처럼 씁니다.

이 코드는 “정수를 담을 수 있는 age라는 상자를 만들고, 그 안에 20을 넣어라”라고 이해하면 됩니다.

age 상자
┌────┐
│ 20 │
└────┘

왜 변수를 써?

점수를 직접 문장에 적으면 점수가 바뀔 때마다 문장 안의 숫자를 찾아 고쳐야 합니다. 변수를 쓰면 값을 한곳에서 관리할 수 있습니다.

int score = 90;
printf("점수는 %d점입니다.", score);

비유하면 변수는 이름표 붙은 보관함입니다. score 보관함 안에 90을 저장해두고, 필요할 때마다 score라고 부르면 안에 있는 값을 가져옵니다.

자료형은 뭐야?

자료형은 상자의 종류입니다. 도시락통에는 밥을 담고, 물병에는 물을 담고, 필통에는 연필을 담듯이 C언어의 변수도 담을 값에 맞는 상자 종류가 필요합니다.

  • int: 정수를 담는 상자
  • float: 소수점을 담는 상자
  • double: 더 정밀한 소수점을 담는 상자
  • char: 문자 하나를 담는 상자

int, float, double, char

int는 1, 2, 10, -5, 100처럼 소수점이 없는 수를 저장합니다. 비유하면 동전 개수, 사람 수, 점수처럼 딱 떨어지는 숫자를 담는 상자입니다.

float는 키 175.5cm, 온도 36.5도처럼 소수점이 있는 값을 담습니다. double도 소수점을 담지만 float보다 더 정밀합니다. 일반 자가 float라면, 더 정밀한 자가 double입니다.

char는 문자 하나를 담습니다. 문자 하나는 작은따옴표를 사용해서 'A'처럼 적습니다.

기억할 점

  • 변수 이름은 숫자로 시작할 수 없습니다.
  • =는 같다는 뜻이 아니라 오른쪽 값을 왼쪽 변수에 저장한다는 뜻입니다.
  • 자료형이 다르면 저장할 수 있는 값의 범위와 출력 서식도 달라집니다.

예제 코드

#include <stdio.h>

int main(void) {
    int score = 92;
    float weight = 62.3f;
    double height = 174.5;
    char grade = 'A';

    printf("점수: %d\n", score);
    printf("몸무게: %.1f\n", weight);
    printf("키: %.1f\n", height);
    printf("등급: %c\n", grade);
    return 0;
}
