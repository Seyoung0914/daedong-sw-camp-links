Lesson 06

반복문

반복문은 같은 작업을 정해진 횟수만큼 또는 조건이 참인 동안 계속 실행합니다.

이전 개념다음 개념

반복문은 뭐야?

반복문은 같은 일을 여러 번 시키는 문법입니다. “안녕”을 100번 출력하려고 printf를 100줄 쓰면 너무 길어집니다. 그래서 반복문을 사용합니다.

반복문은 컴퓨터에게 “이 일을 조건이 맞는 동안 계속해”라고 시키는 방법입니다.

for는 언제 써?

for문은 몇 번 반복할지 정해져 있을 때 많이 사용합니다. 예를 들어 5번 반복, 10번 반복, 100번 반복처럼 횟수가 분명한 경우에 좋습니다.

for (초기식; 조건식; 증감식) {
    반복할 코드
}

처음 보면 어렵지만 for (시작; 언제까지; 어떻게 변할지)라고 생각하면 됩니다.

for 실행 흐름

for (int i = 1; i <= 5; i++) {
    printf("안녕\n");
}

i를 1부터 시작해서, i가 5 이하인 동안 반복하고, 한 번 반복할 때마다 i를 1씩 증가시킵니다. 비유하면 운동장 5바퀴 도는 것입니다.

  • 현재 1바퀴째 → 돈다
  • 현재 2바퀴째 → 돈다
  • 현재 5바퀴째 → 돈다
  • 현재 6바퀴째 → 이제 끝

while은 언제 써?

while문은 조건이 참인 동안 계속 반복할 때 사용합니다. 몇 번 반복할지 정확히 모를 때 많이 씁니다.

  • 사용자가 0을 입력할 때까지 계속 입력받기
  • 비밀번호가 맞을 때까지 계속 입력받기
  • 게임 체력이 0보다 클 때까지 계속 진행하기
int i = 1;

while (i <= 5) {
    printf("안녕\n");
    i++;
}

do while

do while은 일단 한 번은 무조건 실행하고, 그 다음 조건을 검사합니다.

int num;

do {
    printf("숫자를 입력하세요: ");
    scanf("%d", &num);
} while (num != 0);

비유하면 while은 입장권이 있으면 놀이기구를 타는 것이고, do while은 일단 한 번 태우고 다음부터 입장권을 확인하는 느낌입니다.

for와 while 차이

for는 몇 번 반복할지 정해져 있을 때, while은 언제 끝날지 조건에 따라 달라질 때 사용합니다.

  • for: 운동장 5바퀴 뛰어.
  • while: 힘이 남아 있는 동안 계속 뛰어.

기억할 점

  • 반복문의 조건이 계속 참이면 프로그램이 끝나지 않습니다.
  • i++는 반복할 때 카운터를 1씩 증가시키는 표현입니다.
  • 배열과 반복문은 함께 자주 사용됩니다.

예제 코드: 1부터 10까지 출력

#include <stdio.h>

int main(void) {
    for (int i = 1; i <= 10; i++) {
        printf("%d\n", i);
    }

    return 0;
}

i는 번호표처럼 생각하면 됩니다. 번호표 1번부터 10번까지 출력하고, 11번이 되면 조건이 거짓이 되어 반복이 끝납니다.
