Lesson 08

문자와 문자열

문자는 하나의 글자이고, 문자열은 여러 문자를 배열로 묶은 형태입니다.

이전 개념다음 개념

핵심 설명

char는 문자 하나를 저장합니다. 문자 하나는 작은따옴표로 감싸고, 문자열은 큰따옴표로 감쌉니다.

C언어의 문자열은 char 배열로 저장되며, 문자열 끝에는 널 문자 \0이 들어갑니다. 그래서 문자열 배열은 실제 글자 수보다 1칸 더 여유가 필요합니다.

기억할 점

  • 'A'는 문자, "A"는 문자열입니다.
  • 문자 출력은 %c, 문자열 출력은 %s를 사용합니다.
  • scanf("%s", name)처럼 문자열 배열 이름에는 보통 &를 붙이지 않습니다.

예제 코드

#include <stdio.h>

int main(void) {
    char initial;
    char name[20];

    printf("이니셜을 입력하세요: ");
    scanf(" %c", &initial);

    printf("이름을 입력하세요: ");
    scanf("%19s", name);

    printf("이니셜: %c\n", initial);
    printf("이름: %s\n", name);
    return 0;
}
