Lesson 10

포인터

포인터는 값 자체가 아니라 값이 저장된 메모리 주소를 저장하는 변수입니다.

이전 개념목차로

핵심 설명

변수 앞에 &를 붙이면 그 변수의 주소를 얻을 수 있습니다. 포인터 변수는 이 주소를 저장합니다.

포인터 앞에 *를 붙이면 포인터가 가리키는 위치의 값을 읽거나 바꿀 수 있습니다. 이것을 간접 참조라고 합니다.

기억할 점

  • &num은 num의 주소입니다.
  • int *p는 정수형 변수를 가리킬 수 있는 포인터입니다.
  • *p는 p가 가리키는 위치의 실제 값입니다.

예제 코드

#include <stdio.h>

int main(void) {
    int num = 10;
    int *p = &num;

    printf("num의 값: %d\n", num);
    printf("num의 주소: %p\n", (void *)&num);
    printf("p가 가리키는 값: %d\n", *p);

    *p = 20;
    printf("변경된 num의 값: %d\n", num);
    return 0;
}
