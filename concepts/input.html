Lesson 03

입력

입력은 사용자가 프로그램에 값을 전달하는 과정입니다. C언어에서는 주로 scanf를 사용합니다.

이전 개념다음 개념

입력은 뭐야?

입력은 사용자가 키보드로 값을 넣는 것입니다. 출력이 컴퓨터가 사람에게 말하는 것이라면, 입력은 사람이 컴퓨터에게 말하는 것입니다.

  • 출력: 컴퓨터 → 사람
  • 입력: 사람 → 컴퓨터

scanf는 뭐야?

scanf는 사용자가 입력한 값을 변수에 저장하는 함수입니다.

int age;
scanf("%d", &age);

이 코드는 “사용자가 정수를 입력하면, 그 값을 age 상자에 넣어라”라고 이해하면 됩니다.

&는 왜 붙여?

&age는 age 상자의 위치, 즉 주소를 뜻합니다. 컴퓨터에게 “입력값을 age라는 상자에 넣어야 하니까, age 상자가 어디 있는지 알려줄게”라고 말하는 것입니다.

택배 비유로 보면 값은 택배 물건, 변수는 집, &변수는 집 주소입니다. 택배를 보내려면 주소가 필요하듯이, scanf도 입력값을 어디에 넣을지 알아야 합니다.

기억할 점

  • int 입력은 %d, double 입력은 %lf를 사용합니다.
  • 문자 하나를 입력받을 때는 이전 입력의 줄바꿈이 남아 있을 수 있습니다.
  • 문자열 배열 이름은 이미 주소처럼 동작하므로 &를 붙이지 않습니다.

예제 코드

#include <stdio.h>

int main(void) {
    int age;

    printf("나이를 입력하세요: ");
    scanf("%d", &age);

    printf("당신의 나이는 %d살입니다.\n", age);
    return 0;
}

흐름은 이렇게 됩니다. age라는 빈 상자를 만들고, 사용자에게 나이를 입력하라고 말한 뒤, 입력된 값을 age 상자에 넣고 출력합니다.
