Lesson 05

조건문

조건문은 상황에 따라 서로 다른 코드를 실행하게 만드는 문법입니다.

이전 개념다음 개념

조건문은 뭐야?

조건문은 상황에 따라 다른 행동을 하게 만드는 문법입니다. 현실에서도 우리는 매일 조건문을 씁니다. “비가 오면 우산을 쓰고, 비가 안 오면 그냥 간다” 같은 생각이 조건문입니다.

if (비가 온다) {
    우산을 쓴다;
} else {
    그냥 간다;
}

if 기본 구조

if 괄호 안 조건이 참이면 중괄호 안 코드가 실행됩니다.

if (조건) {
    조건이 참일 때 실행할 코드
}

예를 들어 age가 18 이상이면 “성인입니다”를 출력하게 만들 수 있습니다.

if ~ else

else는 조건이 거짓일 때 실행할 코드를 적는 부분입니다. 비유하면 갈림길입니다.

int age = 15;

if (age >= 18) {
    printf("성인입니다.\n");
} else {
    printf("미성년자입니다.\n");
}

나이가 18 이상이면 성인 쪽 길로 가고, 아니면 미성년자 쪽 길로 갑니다.

else if

조건이 여러 개일 때는 else if를 사용합니다. 위에서부터 차례대로 검사하고, 처음 참이 된 부분만 실행합니다.

예를 들어 85점이라면 90점 이상 조건은 거짓이고, 80점 이상 조건은 참이므로 B를 출력하고 끝납니다.

기억할 점

  • 조건식에는 비교 연산자와 논리 연산자가 많이 들어갑니다.
  • 중괄호를 쓰면 실행 범위가 분명해져 실수를 줄일 수 있습니다.
  • if와 else 사이에 불필요한 세미콜론을 넣지 않습니다.

예제 코드

#include <stdio.h>

int main(void) {
    int score;

    printf("점수를 입력하세요: ");
    scanf("%d", &score);

    if (score >= 90) {
        printf("A 등급\n");
    } else if (score >= 80) {
        printf("B 등급\n");
    } else if (score >= 70) {
        printf("C 등급\n");
    } else {
        printf("복습이 필요합니다.\n");
    }
    return 0;
}

이 코드는 점수를 입력받고 60점 이상이면 합격, 그렇지 않으면 불합격을 출력하도록 바꿔 연습해도 좋습니다.
