Lesson 09

함수

함수는 특정 기능을 하는 코드를 이름 붙여 분리한 것입니다.

이전 개념다음 개념

핵심 설명

함수는 입력값인 매개변수를 받을 수 있고, 계산 결과를 반환할 수 있습니다. 함수로 코드를 나누면 같은 기능을 여러 번 재사용하기 쉽습니다.

main도 함수입니다. 프로그램은 보통 main 함수에서 시작하고, 필요한 함수들을 호출하면서 실행됩니다.

기억할 점

  • 반환값이 없는 함수는 void를 사용합니다.
  • 함수 이름, 매개변수, 반환 자료형을 보고 역할을 이해할 수 있어야 합니다.
  • 함수를 호출할 때 전달하는 값의 개수와 자료형이 맞아야 합니다.

예제 코드

#include <stdio.h>

int add(int a, int b) {
    return a + b;
}

void printResult(int result) {
    printf("결과: %d\n", result);
}

int main(void) {
    int x = 3;
    int y = 5;
    int sum = add(x, y);

    printResult(sum);
    return 0;
}
