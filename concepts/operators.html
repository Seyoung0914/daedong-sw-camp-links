Lesson 04

연산자

연산자는 값을 계산하거나 비교하고, 조건을 조합할 때 사용하는 기호입니다.

이전 개념다음 개념

연산자는 뭐야?

연산자는 계산 기호입니다. 수학에서 쓰는 +, -, ×, ÷와 비슷합니다.

연산자의미예시
+더하기a + b
-빼기a - b
*곱하기a * b
/나누기a / b
%나머지a % b

/와 % 차이

10 / 3은 몫을 구합니다. 10을 3으로 나누면 몫은 3이므로 결과는 3입니다. 10 % 3은 나머지를 구합니다. 10을 3으로 나누면 나머지는 1이므로 결과는 1입니다.

비유하면 피자 10조각을 3명에게 똑같이 나눠주는 상황입니다. 각자 3조각씩 먹는 것이 몫이고, 1조각 남는 것이 나머지입니다.

=와 == 차이

C언어에서 =는 수학의 “같다”가 아닙니다. 오른쪽 값을 왼쪽 변수에 넣으라는 뜻입니다. a = 10;은 “a 상자에 10을 넣어라”입니다.

==는 두 값이 같은지 확인하는 비교 연산자입니다. 비유하면 =는 상자에 넣기, ==는 상자 안에 뭐가 들었는지 확인하기입니다.

if (age = 20)   // 잘못된 코드: age에 20을 넣어버림
if (age == 20)  // 올바른 코드: age가 20인지 확인

비교 연산자

연산자의미
==같다
!=다르다
>크다
<작다
>=크거나 같다
<=작거나 같다

C언어에서 비교 결과는 보통 참이면 1, 거짓이면 0으로 나옵니다.

기억할 점

  • ==는 두 값이 같은지 비교하고, =는 값을 저장합니다.
  • 정수끼리 나누면 소수점 아래는 버려집니다.
  • &&는 둘 다 참, ||는 하나만 참이어도 참입니다.

예제 코드

#include <stdio.h>

int main(void) {
    int a = 10;
    int b = 3;

    printf("%d\n", a + b);
    printf("%d\n", a - b);
    printf("%d\n", a * b);
    printf("%d\n", a / b);
    printf("%d\n", a % b);

    printf("%d\n", a >= b);
    return 0;
}
