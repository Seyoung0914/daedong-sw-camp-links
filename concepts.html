C언어는 컴퓨터에게 시키는 심부름

C언어는 어려운 암호가 아니라 컴퓨터에게 아주 정확하게 심부름을 시키는 방법입니다. 변수는 값을 담는 상자, 자료형은 상자의 종류, printf는 화면에 말하기, scanf는 사용자 말 듣기라고 생각하면 시작이 훨씬 쉬워집니다.

개념 쉬운 비유 의미
printf전광판화면에 출력
scanf입력 창구사용자 입력 받기
변수이름 붙은 상자값을 저장

=상자에 넣기값 저장

if갈림길조건에 따라 실행
for정해진 횟수 반복몇 번 반복할 때

초보자가 많이 헷갈리는 것

  • ;는 C언어 문장의 마침표입니다. 문장 끝에는 보통 세미콜론을 붙입니다.
  • { }는 여러 코드를 하나의 묶음으로 만드는 방입니다.
  • ( )는 함수에 필요한 재료나 조건을 담는 접시처럼 생각하면 됩니다.
